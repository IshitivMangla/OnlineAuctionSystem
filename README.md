Auctify is a full-featured online auction platform built with Django, enabling users to buy and sell items securely and efficiently. Users can create accounts, list items for auction, place bids in real-time, and track sold and won items. This platform is designed to provide a seamless online auction experience for both buyers and sellers.

---

## 🛠 Features

- **User Authentication:** Secure registration and login system.
- **Item Listings:** Upload items with images, descriptions, and starting bid prices.
- **Real-Time Bidding:** Place bids and track highest bids dynamically.
- **Auction Management:** Automatic tracking of auction end times.
- **Sold & Won Items:** Users can view items they sold and items they won.
- **Responsive Design:** Works on desktops, tablets, and mobile devices.

---

## 💻 Tech Stack

- **Backend:** Django (Python)
- **Database:** SQLite (or PostgreSQL for production)
- **Frontend:** HTML, CSS, JavaScript
- **Optional Libraries:** Bootstrap for UI styling, jQuery for dynamic updates
